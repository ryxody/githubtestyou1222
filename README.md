# githubtestyou1222
На данном репозитории я бы хотел рассказать о проекте Arduino IDE эмуляции клавиатуры через электропроводник:
1. Нам нужен сам arduino 
2. Нужен код оптимизированный под эмуляцию и провода с электропроводником

В роли электропроводника подойдёт:
Картошка, Алюминий и т.д

3. Скачать приложение Arduino IDE (https://www.arduino.cc/en/software) и инициализировать устройство
4. Загрузить представленный код выше на Arduino
5. Подключить в правильном порядке устройство для эмуляции (Соединить Arduino с проводником к примеру банкой)

   Принцип работы:
Так как человек проводит ток - при прикосновении к банке или картошке и подсоединенным Arduino на плату поступает сигнал и устройство считает нажатием так и работает устройство
